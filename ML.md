## 机器学习笔记
### 性能指标
- 准确率 = （正确样本数 / 总样本数）
- 非均衡问题
  
    |预测/实际| 1       |       -1   |
    | :---:  | :---:    | :---:       |
    |   1    |TP（真阳）|  FP（假阳） |
    |   -1   |FN（假阴）|  TN（真阴） |
    查准律(Precision)= TP / (TP + FP)
    
    查准率相当于“挑出来的西瓜里有多少好的”
    
    召回率(Recall) = TP / (TP + FN)

    召回率相当于“好西瓜有多少被挑出来”
